Process

"ACTinG for U" consists of a water system/environment and a five-computer cluster. The water system is controlled by an Arduino that runs four peristaltic pumps across three colored liquids and one backup water container, all into one main narrow container. On top of the main container, there are three sensors (a PIR sensor, long-range ultrasonic sensor, and short-range ultrasonic sensor) to detect various external environmental aspects that influence the system behavior as a metaphorical organism. The Arduino is powered by one of the computers in the five-computer cluster which doubles as a communication channel via the serial port. The cluster visualizes the gene sequences in the human genome onto eight screens in a two-by-four set-up. The visualization changes over time through a cyclical animation, programmed in Python with the MPI4py and Pygame libraries. See Behavior page for more information on the internal code.
